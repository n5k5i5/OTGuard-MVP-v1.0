# Возможности (RU)

Этот проект — безопасный каркас (только CLI) для тестирования безопасности с приоритетом лабораторной среды. Ниже текущее состояние и план на v2.0.

## Текущее (v1.0)

- Модульная архитектура и SDK для модулей
  - Разработка модулей через manifest + Python-класс
  - Примеры (безопасные для лаборатории): Port Scan, System Info, импорт Nmap XML
- Автоматизация (Resource DSL)
  - YAML-шаги, переменные, условия (`when`), циклы (`foreach`)
  - Интерполяция и динамические псевдонимы
- Отчеты и метрики
  - JSON/HTML на запуск; индекс запусков (.runs/index.html)
  - База SQLite для метрик (.runs/metrics.db)
  - Логи CLI в формате JSONL (.runs/app.log)
- Сессии (заглушки)
  - Локальные и Docker-сессии
- Управление и безопасность
  - Принятие EULA (EN/TR/RU)
  - Guardrails: лабораторный режим, блок публичных IP
  - Шлюз согласования организации (OrgGate)
  - Опциональный второй фактор для отчетов (ReportGate)
- Пакетирование и дистрибуция
  - GitHub/PyPI workflow; встроенные пример-модули
- Документация
  - Быстрый старт, руководство разработчика модулей, архитектура, политика (OrgGate)
